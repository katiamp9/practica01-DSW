# Frontend Empleados .

Base de frontend para integración con backend Spring Boot.

## Estructura

- `src/app/componentes/`
- `src/app/servicios/`
- `src/app/modelos/`
- `src/app/interceptores/`

## Desarrollo

1. Instalar dependencias: `npm install`
2. Ejecutar desarrollo: `npm start`
3. El proxy reenvía `/api` a `http://localhost:8080`

## Estándares

- Standalone Components
- Signals
- Control Flow nativo
